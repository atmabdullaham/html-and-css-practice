Practice of font-family

In CSS there are five generic font families:

Serif fonts have a small stroke at the edges of each letter. They create a sense of formality and elegance.

Sans-serif fonts have clean lines (no small strokes attached). They create a modern and minimalistic look.

Monospace fonts - here all the letters have the same fixed width. They create a mechanical look.

Cursive fonts imitate human handwriting.

Fantasy fonts are decorative/playful fonts.

Practice of font-size

The font-size property sets the size of the text.

The font-size value can be an absolute, or relative size.

Absolute Size:

  • Sets the text to a specific size
  • Does not allow a user to change the text size in all browsers
  • Absolute size is useful when the physical size of the output is known

Relative Size:

  • Sets the size relative to surrounding elements
  • Allows a user to change the text size in browsers

Sets font size with em

To allow users to resize the text (in the browser menu), many developers use em instead of pixels. 1em is equal to the current font size. The default text size in browsers is 16px. So, the default size of 1em is 16px. The size can be calculated from pixels to em using this formula: *pixels*/16=*em*

CSS provides various measuring units to specify sizes, lengths, and dimensions. Here’s a breakdown of the most common CSS measuring units along with examples:

Absolute Units

Absolute units are fixed in size and do not change with screen size or resolution.

Relative Units

  1. ### a. Percentages (%) - **Definition**: A percentage is relative to the parent element's size. - **Use Case**: Commonly used for responsive design.
  2. ### b. Em (em) - **Definition**: Relative to the font size of the element (1em = the current font size). - **Use Case**: Useful for scalable layouts.
  3. ### c. Rem (rem) - **Definition**: Relative to the root element's font size (1rem = font size of the root element). - **Use Case**: Provides consistency across different elements.
  4. ### d. Viewport Units (vw, vh, vmin, vmax) - **Definition**: Relative to the viewport size. - **vw**: 1% of the viewport width. - **vh**: 1% of the viewport height. - **vmin**: The smaller value of vw or vh. - **vmax**: The larger value of vw or vh. - **Use Case**: Great for responsive design.
